# 02 — User Personas

---

## Persona 1: Rajesh — Property Owner

| Attribute | Detail |
|-----------|--------|
| **Age** | 42 |
| **Role** | Owns 3 PGs in Bangalore (120 beds total) |
| **Tech** | Uses WhatsApp, Excel, basic smartphone |
| **Goals** | Maximize occupancy, reduce rent defaults, scale without hiring more managers |
| **Pain** | No visibility across properties; managers hide problems; manual UPI reconciliation |
| **Rentle Value** | Unified dashboard, automated rent reminders, collection reports, white-label tenant app |
| **Key Screens** | Owner dashboard, revenue analytics, property list, subscription billing |

---

## Persona 2: Priya — Property Manager

| Attribute | Detail |
|-----------|--------|
| **Age** | 28 |
| **Role** | Manages one 40-bed PG for Rajesh |
| **Tech** | Power user of mobile apps |
| **Goals** | Fill vacant beds fast, resolve complaints, coordinate staff |
| **Pain** | Leads in 5 WhatsApp groups; room status outdated; tenant disputes on rent |
| **Rentle Value** | CRM kanban, bed-level availability, complaint tickets, notice broadcast |
| **Key Screens** | Manager dashboard, leads kanban, rooms/beds grid, complaints inbox |

| Attribute | Detail |
